# 通貨コンバーター | Currency Converter

A real-time currency converter with an anime-inspired neon cyberpunk design.

![Anime Currency Converter](https://img.shields.io/badge/style-neon%20cyberpunk-ff2a6d?style=for-the-badge)
![Currencies](https://img.shields.io/badge/currencies-RUB%20%7C%20USD%20%7C%20KZT%20%7C%20JPY-9d4edd?style=for-the-badge)

## Features

- **Real-time exchange rates** — fetched from Frankfurter API
- **Instant conversion** — type in any field, others update automatically
- **4 currencies** — Russian Ruble, US Dollar, Kazakhstan Tenge, Japanese Yen
- **"Lucky" button** — generates random amounts for fun
- **Auto-refresh** — rates update every 5 minutes
- **Offline mode** — uses cached rates when offline
- **Responsive design** — works on mobile and desktop

## Design

- Neon cyberpunk color palette (purple, pink, cyan)
- Japanese-inspired decorative elements (torii gate, Mt. Fuji silhouettes)
- Anime poster fonts (Dela Gothic One, Orbitron)
- Glow effects and smooth animations

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, animations)
- Vanilla JavaScript (ES6+)
- [Frankfurter API](https://www.frankfurter.app/) for exchange rates

## Usage

1. Open `index.html` in a browser
2. Enter an amount in any currency field
3. Watch other currencies convert automatically
4. Click "Мне повезёт!" for random amounts

## Local Development

Simply open `index.html` in your browser. No build tools required.

```bash
# Clone the repository
git clone https://github.com/Cosmovisioner/Converter.git

# Open in browser
open index.html
```
